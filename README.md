# Online Shoppers' Purchasing Intentions (Excel)

**The Data:** The Online Shoppers Purchasing Intention dataset is a collection of data related to purchase patterns and consumer behavior in the context of online shopping. It was created by conducting surveys of online shoppers and collecting data from their responses ([UC Irvine](https://archive.ics.uci.edu/dataset/468/online+shoppers+purchasing+intention+dataset)).
The Excel file contains sheets regarding notes about the dataset, the original dataset, a modified version of the dataset, analyses, & a final dashboard ("Dashboard").

**Purpose:** An Excel dashboard was created to display relevant metrics according to the variables involved such as number of sessions, purchase groups, & more.
